# aviutl2-text-parser

AviUtl2のテキスト記法のパーサー。

<https://docs.aviutl2.jp/usage> のテキスト制御文字仕様に追従しています。

現在は以下の制御文字をパースできます。

- `<#...>`: 色変更
- `<s...>`: サイズ、フォント、装飾、縁取りサイズ変更
- `<@...>`, `<@+...>`, `<@-...>`: フォント、文字装飾変更
- `<$...>`: プリセット適用
- `<r...>`: 表示速度変更
- `<w...>`: 表示ウェイト
- `<c...>`: 表示クリア
- `<p...>`: 座標指定
- `<gw...>`, `<gh...>`: 字間、行間変更
- `<tw...>`, `<th...>`, `<tr...>`: 文字変形
- `</>...<!...>...</>`: ルビ指定
- `</>`: 文字列ブロック区切り
- `<//...//>`: コメント
- `<?...?>`, `<?=...?>`: スクリプト
